#  Кинокаталог (ДЗ 12)

Полнофункциональное React-приложение с маршрутизацией, загрузкой данных из OMDb API и управлением состоянием через Context API.

> **Дисциплина:** Разработка прототипов программных решений  
> **Тема:** Основной и продвинутый React. Компоненты, состояние и хуки  
> **Преподаватель:** Марк Садыков

---

[Открыть приложение](https://react-shop-dz12.vercel.app)


##  Структура проекта

src/
├── pages/ # Страницы приложения

│ ├── Home.tsx # Главная страница

│ ├── List.tsx # Каталог товаров

│ ├── Details.tsx # Детали товара

│ ├── About.tsx # Страница "О нас"

│ └── Favorites.tsx # Избранное

├── components/ # Переиспользуемые компоненты

│ ├── Navbar.tsx # Навигационное меню

│ ├── ProductCard.tsx # Карточка товара

│ ├── Spinner.tsx # Индикатор загрузки

│ └── ErrorMessage.tsx# Сообщение об ошибке

├── context/ # Глобальное состояние

│ └── AppContext.tsx # Context API (избранное + кэш)

├── services/ # Слой работы с сетью

│ └── api.ts # Fetch-запросы к API

├── types.ts # TypeScript-интерфейсы

├── App.tsx # Настройка маршрутизации

└── main.tsx # Точка входа
```

### Задание 1: Маршрутизация
- [x] Настроены маршруты: `/`, `/list`, `/list/:id`, `/about` через React Router v6
- [x] Навигация через `<NavLink>` с автоматической подсветкой активной ссылки
- [x] Созданы компоненты страниц: `<Home />`, `<List />`, `<Details />`, `<About />`

### Задание 2: Работа с сетью
- [x] Загрузка списка товаров из публичного API (FakeStoreAPI)
- [x] Загрузка деталей товара по ID через динамический маршрут `/list/:id`
- [x] Индикатор загрузки (`<Spinner />`) и обработка ошибок (`<ErrorMessage />`)
- [x] Кнопка «Повторить» при сбое запроса

### Задание 3: Управление состоянием
- [x] Context API для глобального состояния (избранное + кэш товаров)
- [x] Кэширование данных: нет повторных запросов при переходе между страницами
- [x] Сохранение избранного в `localStorage` (не сбрасывается при обновлении)
- [x] Кнопка «В избранное» на странице товара и в карточке

##  Технологии

| Инструмент | Назначение |
|------------|-----------|
| React 18 + TypeScript | Фронтенд-фреймворк + типизация |
| Vite | Сборка и dev-сервер |
| React Router DOM v6 | Маршрутизация |
| Context API + localStorage | Глобальное состояние |
| Fetch API | Сетевые запросы |
| ESLint + Prettier | Линтинг и форматирование |

##  Шаг 2: Финальная проверка (30 секунд)

Прокликай быстро:
1. `/list` → фильмы грузятся ✅
2. Клик «Подробнее» → открывается страница с постером и описанием ✅
3. Нажми ❤️ на любом фильме → счётчик в меню обновился ✅
4. Перейди в `/favorites` → фильм отображается ✅
5. Нажми `F5` → избранное **не пропало** ✅
6. В консоли браузера (`F12`) **нет красных ошибок** ✅

Если всё ок → переходим к коммиту.
